# Tri-NIT

___
## Setup

* ### Requirements
    * [Node](https://nodejs.org/en/)
    * [Go](https://go.dev/)
    * [Docker](https://www.docker.com/)


* ### Run
    * ##### Admin-Service
        ```bash
        cp .env.example .env && docker-compose up
        ```
    * ##### Dashboard
        ```bash
        npm i && npm run dev
        ```
